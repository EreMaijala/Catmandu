=head1 Catmandu

=head2 Building Catmandu

Catmandu uses L<Dist::Zilla> as build tool.

Make sure you have the following CPAN modules installed:

    App::cpanminus
    Dist::Zilla
    Dist::Zilla::Plugin::OurPkgVersion
    Dist::Zilla::Plugin::PodWeaver

Use the C<dzil> command:

    dzil build
    dzil test
    dzil clean
    dzil release
    ...

Install project dependencies:

    dzil listdeps | cpanm

=cut
